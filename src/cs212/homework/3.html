To start, I looked online to find free fonts which I could use for this lab. After looking at the site dafont.com, I found a font called "INFECTED" which I liked. From there, I downloaded the zip file which I would then move into my GitHub static web app.

Now in my web app, I entered the homework folder where I would then make another folder titled "css" where a "styles.css" file would be. Within the "css" folder, I placed the "INFECTED" font file so that my "styles.css" file could refer to it using "@font-face". Within the "styles.css" file, I used "@font-face" to refer to the "INFECTED" file, and used a rule to make all text within the body to have the "INFECTED" font. To make it easier to see, I also made all text have a font-size of 40px.
